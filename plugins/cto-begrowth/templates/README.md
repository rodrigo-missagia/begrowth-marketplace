# Templates do CTO Plugin

Este diretorio contem os templates usados para inicializar a estrutura de knowledge.

## Arquivos

- `holding-index.md` - Template para holding/_index.md
- `holding-stack.md` - Template para holding/stack.md
- `holding-sinergias.md` - Template para holding/sinergias.md
- `empresa-index.md` - Template para [empresa]/_index.md
- `empresa-contexto.md` - Template para [empresa]/contexto.md
- `pessoas-index.md` - Template para */pessoas/_index.md
- `iniciativas-index.md` - Template para */iniciativas/_index.md
- `adrs-index.md` - Template para */adrs/_index.md

## Variaveis de Substituicao

Os templates usam variaveis que sao substituidas durante a inicializacao:

- `{{EMPRESA}}` - Nome da empresa (utua, resolve, etc)
- `{{EMPRESA_UPPER}}` - Nome em maiusculas (UTUA, RESOLVE, etc)
- `{{EMPRESA_DESCRICAO}}` - Descricao da empresa
- `{{DATA}}` - Data atual (YYYY-MM-DD)
